# LD_PRELOAD

原理：

PHP 的 putenv()函数，设定 LD_PRELOAD(环境变量) 为 hack.so。

利用 PHP 的 mail()函数，mail() 内部启动新进程 /usr/sbin/sendmail，因为上一步 LD_PRELOAD的作用，sendmail 调用的void()函数 被优先级更好的 hack.so 中的同名 getuid()函数所劫持。
